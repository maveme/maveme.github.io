üìä Interpretation
  • Shapiro-Wilk p-value > 0.05: Data can be considered normally distributed
  • Shapiro-Wilk p-value <= 0.05: Data significantly deviates from normal distribution
  • Warning: Results for small sample sizes (n < 8) should be interpreted with caution

Note: The "All Periods" analysis combines data across all periods to provide a more comprehensive assessment with larger sample sizes.

üìà QQ Plots

QQ plots compare the observed data distribution to the theoretical normal distribution:

  • Points following the red reference line closely suggest a normal distribution
  • Systematic deviations from the line suggest non-normality
  • S-shaped patterns indicate skewness in the data
üîç KDE Plots

Kernel Density Estimation (KDE) plots show the estimated probability density of recovery times:

  • Bell-shaped curves suggest normal distributions
  • Multi-modal curves (multiple peaks) suggest mixed distributions
  • Skewed curves (asymmetric) indicate non-normal distributions
  • Dotted line shows theoretical normal distribution for comparison
  • Green vertical line shows mean, blue vertical line shows median
